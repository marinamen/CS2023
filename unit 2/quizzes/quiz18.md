18. Create a function that produces the table of Truth for 3 inputs:

　　　.   　　˚　　 　　*　　 　　✦　　　.　　.　　　✦　˚ 　　　　 ˚　.˚　　　　✦

*code*

     def get_truth():
         print("A | B | C |")
         input = [0, 1]
         for a in input:
             for b in input:
                 for c in input:
                     print(f"{a} | {b} | {c} |")
     
     get_truth()
